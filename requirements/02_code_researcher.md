wir brauchen researcher, denen der architekt aufgaben geben kann

der architekt darf kurzen überblick per eingebautem explore agent nutzen, bei tieferen analysen in großen code bases es soll code reaseracher agenten nutzen. Diese sollen auf basis haiku einen reasearch auftrag bekommen und in der codebase sowie dokumentation ein eine analyse der architektur erstellen. er erzeugt 2 unterschiedliche artefakte:

- high level summary for the architect - answering all questions the architect had
- a detailed research that is intended for design or coding agents
